# rust-youtube-dl
## About the project
Youtube Downloader using Rust
## Getting Started
### Prerequisites
- Rust v1.43.0 or higher
```
git@github.com:coska/rust-youtube-dl.git
cd rust-youtube-dl
cargo build --release
target/release/rust-youtube-dl YOUTUBE-VIDEO-URL
```
